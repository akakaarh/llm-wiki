# 数字逻辑基础

数字电路是嵌入式硬件的底层基础。理解逻辑门、触发器和时序逻辑，有助于阅读芯片数据手册中的时序图、理解外设寄存器的工作原理。

---

## 布尔代数

### 基本运算

| 运算 | 符号 | C 表达式 | 真值表（A=1,B=0 为例） |
|------|------|----------|----------------------|
| AND | `A · B` | `A & B` | 0 |
| OR | `A + B` | `A \| B` | 1 |
| NOT | `Ā` | `~A` | 0 |
| XOR | `A ⊕ B` | `A ^ B` | 1 |
| NAND | `(A · B)̄` | `~(A & B)` | 1 |
| NOR | `(A + B)̄` | `~(A \| B)` | 0 |

### 基本定律

- **交换律**：`A+B = B+A`, `A·B = B·A`
- **结合律**：`(A+B)+C = A+(B+C)`
- **分配律**：`A·(B+C) = A·B + A·C`
- **德摩根定律**：`A̅·B̅ = (A+B)̅`, `A̅+B̅ = (A·B)̅`
  - 嵌入式应用：NAND/NOR 门是制造工艺中最基础的门，任何逻辑都可以用 NAND 门组合实现

### 逻辑门符号

```
AND:        OR:         NOT:        XOR:        NAND:
  A─┐         A─┐        A─○─Ā       A─┐         A─┐
    ├─Y         ├─Y                   ├─Y         ├─○─Y
  B─┘         B─┘        (三角+圈)   B─┘         B─┘
```

## 组合逻辑（Combinational Logic）

输出仅取决于当前输入，无记忆功能。

### 多路选择器（MUX）

从多个输入中选择一个输出，由选择信号控制。

```
2:1 MUX:
  I0─┐
    ├─[MUX]─Y
  I1─┘
    S (选择线)

Y = I0 when S=0, Y = I1 when S=1
C 表达式：Y = S ? I1 : I0
```

应用：CPU 中的 ALU 操作选择、外设寄存器的位域选择。

### 解码器（Decoder）

n 位输入 → 2ⁿ 位输出，仅一个输出为高。

```
2-to-4 解码器：
输入 A[1:0] → 输出 Y[3:0]
  00 → 0001
  01 → 0010
  10 → 0100
  11 → 1000
```

应用：内存地址译码、外设片选信号生成。

### 编码器（Encoder）

与解码器相反，2ⁿ 位输入 → n 位输出。

优先编码器（Priority Encoder）：多个输入同时有效时，输出最高优先级的编码。中断控制器中常用。

## 时序逻辑（Sequential Logic）

输出取决于当前输入**和**之前的状态，具有记忆功能。

### 触发器（Flip-Flop）

触发器是时序逻辑的基本存储单元，在时钟边沿触发。

#### D 触发器（D Flip-Flop）

最常用的触发器类型。

```
       ┌───┐
  D ──►│   │── Q
       │ D │
CLK ──►│   │── Q̅
       └───┘

功能：在 CLK 上升沿，Q = D
```

- **建立时间（setup time）**：D 信号必须在 CLK 边沿前稳定的最小时间
- **保持时间（hold time）**：D 信号必须在 CLK 边沿后保持稳定的最小时间
- 这两个参数决定了电路的最高工作频率（时序约束）

#### JK 触发器

```
J=0, K=0 → 保持
J=0, K=1 → 清零
J=1, K=1 → 翻转（toggle）
J=1, K=0 → 置位
```

应用：计数器、分频器。

#### T 触发器

JK 触发器的特例（J=K=T），T=1 时每个时钟沿翻转。用于计数器和分频。

### 寄存器（Register）

多个 D 触发器并排，共用同一个时钟信号，构成寄存器。

```
D[3:0] ──►[D FF][D FF][D FF][D FF]── Q[3:0]
           ↑
          CLK

功能：在 CLK 上升沿，锁存4位数据
```

- CPU 中的通用寄存器（R0~R15）就是32个 D 触发器并排
- 外设的状态寄存器也是寄存器，读取的是触发器的当前状态

### 移位寄存器（Shift Register）

```
Din → [FF] → [FF] → [FF] → [FF] → Dout
       ↑CLK   ↑CLK   ↑CLK   ↑CLK

每个时钟沿，数据向右移动一位
```

应用：SPI 的数据移位、UART 的串并转换。

## 计数器（Counter）

由触发器和组合逻辑构成，按时钟节拍递增/递减。

### 异步计数器（Ripple Counter）

每个触发器的输出作为下一个触发器的时钟，逐级传播。

- 优点：电路简单
- 缺点：存在传播延迟，高频下各触发器状态不同步

### 同步计数器

所有触发器共用同一个时钟，通过组合逻辑决定翻转条件。

- 优点：无传播延迟，工作频率高
- 缺点：组合逻辑复杂度随位数增加

应用：定时器的预分频器（Prescaler）就是计数器。

## 在嵌入式中的应用

| 数字逻辑概念 | 嵌入式中的体现 |
|-------------|---------------|
| D 触发器 | CPU 寄存器、外设数据寄存器 |
| 计数器 | 定时器（TIM）、看门狗（WDG） |
| MUX | GPIO 复用（AFR 寄存器选择外设功能） |
| 解码器 | 内存映射地址译码、外设片选 |
| 优先编码器 | 中断优先级仲裁（NVIC） |
| 移位寄存器 | SPI 数据收发、UART 串并转换 |
| 三态门 | 总线共享（多个设备挂同一总线，未选中时输出高阻态） |
