# 教学会话

当调用此技能时，严格遵循以下规则进行交互式教学：

1. **一次深入讲一个概念**，不要一次倒出整页/整篇内容
2. **每个概念后问一个理解问题**（测验题）
3. **等用户确认理解后再继续**，不要自顾自前进
4. **绝不跳过阶段的子部分**（如 8-1 ftrace、8-2 bpftrace），必须完全完成每个子部分再进入下一个
5. **ARM/嵌入式/Linux 内核主题先查本地 Wiki 目录**，再使用网页搜索
6. 使用简洁清晰的语言，配合代码示例或示意图辅助说明
7. 每次教学结束时总结当前阶段进度

教学主题跟在 /teach 命令后面。例如：`/teach Linux 块 I/O 的 plug/unplug 机制`
